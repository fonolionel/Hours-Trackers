About This Project

This project is a Teaching Hours Tracker application built using Flask, a web framework for Python. It is designed to help teachers manage and track their teaching hours efficiently.

Features

  • Track total teaching hours and hours taught by each teacher.
  • Validate and update teaching hours after classes are conducted.
  • User-friendly interface for teachers to input and manage their teaching hours.

Project Structure

The project consists of the following main components:

  • app.py: The main application file containing Flask routes and logic.
  • templates/: HTML templates for rendering the web pages.
  • static/: Static files such as CSS and images.
  • data.json: A JSON file for storing teaching hours data.

Getting Started

To run this application locally, follow these steps:

  1. Set up a virtual environment and activate it.
  2. Install Flask using pip.
  3. Run the Flask application using python app.py.
  4. Open a web browser and navigate to http://127.0.0.1:5000.
